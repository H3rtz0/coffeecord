# CoffeeCord

一个轻量的咖啡冲煮记录小程序（静态页面版本）。

## 功能

- 新增冲煮记录（咖啡豆、方式、粉水比、评分、备注）
- 编辑已有记录（回填表单并保存修改）
- 历史记录筛选（方式、最低评分、关键词）
- 历史记录排序（时间/评分，升序/降序）
- 自动计算平均评分，并显示当前筛选结果条数
- 数据导出（JSON）
- 数据导入（JSON，支持合并去重/覆盖）
- 导入时自动过滤无效记录并给出结果提示
- 本地持久化（`localStorage`）
- 删除单条 / 清空全部

## 本地运行

```bash
python3 -m http.server 4173
```

然后打开 <http://localhost:4173>。
